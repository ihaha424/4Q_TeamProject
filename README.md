# 시나브로 (Sinabro)

게임인재원 6기 4Q 팀 프로젝트

## 📝 프로젝트 소개

시나브로는 게임인재원 6기에서 진행한 4분기 팀 프로젝트입니다. 
자체 제작 게임 엔진과 네트워크 기능을 중심으로 개발되었습니다.

## ✨ 주요 기능

- **자체 게임 엔진**: 직접 개발한 커스텀 게임 엔진
- **네트워크 시스템**: 멀티플레이어 지원을 위한 네트워크 구현

## 🛠 기술 스택

- **언어**: C++
- **그래픽스**: DirectX 11
- **물리 엔진**: PhysX
- **네트워크**:  Protobuf
- **사운드**:  FMOD
